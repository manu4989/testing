Connect

Connect is the default screen that welcomes you to Pro Fit offering two programming options. This screen displays information about the detected hearing aids and provides options for working with the aids, and TeleHear loads a fitting session to adjust hearing aids remotely. In addition, patients and audiograms can be viewed and maintained via the Noah or PatientBase database. Follow the instructions below for Connected Devices and see TeleHear for more information on this feature.

 

There are two options for working with the detected hearing aids: Connect and Performance Update (if available). Connect starts a new fitting session using the settings in the hearing aid and Performance Update updates the hearing aid firmware to operate at a more advanced performance level or upgrades and/or downgrades the product's tier. To retrieve a previously saved fitting session and transfer the settings into the connected hearing aid, see Retrieve Session.

 

Detecting Devices

 

Pro Fit automatically reads and identifies all hearing aids connected to any programmer (Starkey 2.4 GHz Wireless, Noahlink Wireless, SpeedPort 2, SpeedPort, NOAHlink, USB Hi-Pro 2, or USB Hi-Pro) selected via Programmers and displays the information on the screen. For RIC devices, the type (stock or custom), matrix, and side are automatically detected if a receiver cable is attached. For previously fit devices, if the receiver cable matrix and side don't match the matrix and side in the aid, a message displays.

 

If Pro Fit detects hearing aids that require using Inspire software to program the devices, a message displays. Click Open Inspire to either launch the software or install it.

  1. Place the wireless hearing aid(s) within range of the Starkey 2.4 GHz Wireless or Noahlink Wireless Programmer or connect the desired wired hearing aid(s) using the appropriate boot and/or cable. All the hearing aids connected to each programmer displays. Aids detected for the patient chosen in Noah or PatientBase are listed first and automatically selected.
    Note:  For best results, program with a fully charged or fresh battery inserted into the hearing aid(s). Rechargeable batteries must be removed before programming wired hearing aids. Wireless hearing aids must have a battery inserted for wireless programming.

  2. Click the hearing aid(s) to program, if applicable.
    Tip:  To locate a specific wireless hearing aid, see Wireless Programming.
    Note:  For 2.4 GHz CROS/BiCROS devices, once the receiver and transmitter are paired, only the receiver needs to be connected for adjustments. Regardless if just the receiver or both are connected, only information for the receiver displays on most screens.
  3. Click Connect or Performance Update (if available).
    Note:  If the patient's first and last names selected in Noah or PatientBase don't match the patient's first and last names in the hearing aid(s) exactly (case sensitive not applicable), a message displays. Click Yes to change the patient's name in the aid(s) or click No to cancel the connection.
    Note:  If the hearing aid(s) contains a lock code and it doesn't match the code via General Preferences, it must be entered to unlock the aid before it can be read and adjustments made. Enter the 4 digit code and click OK. If the correct code is entered or if the code matched the code via General Preferences initially, the hearing aid is automatically unlocked and must be locked again if desired via Device Access Settings. If the code is unknown, click Cancel to cancel the fitting and return to the Connect screen or click Clear Settings and Reset Device to clear all settings and patient data from the hearing aid and return it to the default settings.
    Note:  If only one wireless hearing aid was selected that is paired with another aid that was not selected or connected, a message displays to connect the partner aid. Click OK to fit monaurally or click Cancel to cancel the fitting and return to the Connect screen.
    Note:  Pro Fit requires a minimum of four audiometric thresholds at 500, 1000, 2000, and 4000 Hz to fit adult patients and a minimum of two thresholds between 250 and 6000 Hz to fit pediatric patients. If any of these thresholds are missing on the patient's audiogram, Pro Fit automatically predicts the missing data and provides the option to either use this predicted data or edit the audiogram. If there's a conflict between the audiogram in the hearing aid and Noah or PatientBase, a dialog box displays asking which audiogram to use. Click Hearing Aid to use the hearing aid's audiogram or click Office System to use Noah or PatientBase's audiogram. If there's a conflict between the fitting formula in the hearing aid and the default fitting formula selected in Pro Fit, a dialog box displays asking which fitting formula to use. Click on the radio button representing the desired fitting formula to use and if the option selected should apply to all future conflicts, click the checkbox Remember my preference and don’t ask me again to bypass this message in the future. Click Continue to use the selected fitting formula or click Cancel to cancel the fitting and return to the Connect screen. If the default fitting formula was selected, a Best Fit All Programs is automatically performed on the applicable hearing aid(s).
    Note:  If paired wireless hearing aids were selected that don't have matching wireless capabilities and the same configuration, the Wireless Settings Conflict dialog displays asking you to select which device has the desired functionality for your fitting. Click Select for the source hearing aid and Pro Fit automatically changes the settings in the other hearing aid to match those of the selected hearing aid or click Cancel to cancel the fitting and return to the Connect screen. See Wireless Programming for more information.
    Note:  If the acoustic options have not been set on applicable hearing aids, the Acoustics screen automatically displays.
  4. Program the hearing aid(s).
    Note:  If programming a wireless hearing aid, the physical user control, if applicable, is disabled during the fitting. Once the fitting session is complete, the physical user controls can be re-enabled to test the hearing aid without exiting Pro Fit. See Wireless Programming for more information.
    Note:  As the hearing aid(s) is programmed, the connection status icons will change.
    Note:  If programming a wireless hearing aid and the connection is lost during the fitting, Pro Fit automatically attempts to reestablish connection for 15 seconds. If this is unsuccessful, a message displays indicating the specific side(s) that went out of range and to reestablish connection. See Wireless Programming for more information.
